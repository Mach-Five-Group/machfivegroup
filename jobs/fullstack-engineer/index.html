About the Role

Join Mach Five Tech as a Full Stack Engineer where you'll embed directly with client teams to build high-velocity, sustainable software. We're not just another dev shop - we partner deeply with early-stage and growth companies, moving from concept to traction in weeks, not months. You'll work with modern web technologies (Core Web APIs, Node.js, Vanilla JavaScript) to create pragmatic solutions that solve real business problems. Our engineers don't hide behind tickets - you'll collaborate directly with stakeholders, make architectural decisions that matter, and ship meaningful features rapidly. We value clean, maintainable code over trendy frameworks, and believe in building deployment confidence through incremental rollouts and robust monitoring. If you thrive in an outcome-driven environment where your technical decisions directly impact client success, this role offers the perfect blend of technical challenge and business impact.

Duties

  • Develop core application components
  • Adhere to and enforce the practices and procedures outlined in the team's working agreement
  • Perform peer code reviews
  • Provide time estimates for the development of software components

Requirements

  • 3+ years building production web applications
  • Deep experience with Core Web Technologies, Node.js, Vanilla JavaScript
  • Database design (PostgreSQL, MongoDB)
  • API design and integration
  • Client communication skills
  • Cloud platform experience (AWS, Azure, GCP)

Desired Knowledge Areas

  • Advanced Responsive Web Design CSS
  • async REST (Fetch)
  • Familiar with Figma
  • Git
  • Pinia
  • Playwright End-to-End Automated Testing
  • SSO / JWT
  • Web Components
